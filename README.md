<h1>Photo Website</h1>
<p>My first "big" project created to learn react and react router. The project shows the website of a sample photography company.</p>

<h2>Project is created with:</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Swiper JS</li>
  <li>React</li>

 </ul>
 
<h3>Run</h3>
<p>Run project using npm: </p>

```

npm install

```
